# Cloud MLM Software

Cloud MLM Software is a software package that can be used to manage Multilevel marketing business.


## Requirements

* PHP >= 7.2.0 

* BCMath PHP Extension

* Ctype PHP Extension

* JSON PHP Extension

* Mbstring PHP Extension

* OpenSSL PHP Extension

* PDO PHP Extension

* Tokenizer PHP Extension

* XML PHP Extension
